- 시간: **저녁(18-23) 피크 + 새벽(0-5) 두꺼움** = 하루 마무리 회고
- "R&B·인디 좋아하지만 곡은 잘 모름" → 발견받는 재미로 사용

### 학생 비중
- 기기 기준 **약 8%** (18/190, '야자수' 오매칭 제외). 교복·교실·복도·1-5반·쉬는시간 신호 선명. 중·고생 = 타깃 확장 + 또래 바이럴층(구매력 낮음).

## 5. 사용 행동 = "가챠 재미 + 미리듣기 머신" (검증됨)

- 분석의 **64.5%**에서 미리듣기 → 단 **완청 12%**(88% 스킵) = 빠른 맛보기.

- → "진지한 음악 큐레이션"이 아니라 **"사진 넣고 곡+vibeType 뽑아보는 가챠 + 30초 곡 맛보기"**. 얕아서 가챠 피로 시 이탈(D7 1.3%).

## 8. 레퍼런스 앱 (왓챠 / 마인드브릿지)

- **왓챠피디아 = "발견 엔진" 모델**: 평가 데이터 축적 → 추천 정확도 → 플라이휠(소비 70%가 추천 경유). 플더픽도 저장/완청을 취향 신호로 학습 + "내 음악 취향 리포트"(자기표현).
- **마인드브릿지 = "retention 후크" 모델** (마음 질문 다이어리 앱): 매일 질문 + **펫 성장 게임화** + 이모지 다이어리 + 친구 공유. → 플더픽의 **픽터를 "사진 올릴수록 자라는 펫"으로** + 오늘의발견 일일화 + 음악일기 축적. (지인의 "픽터 옛날 느낌" 피드백도 픽터 역할 재정의로 해결)
- 주의: 사진은 질문보다 무거운 행동 → 매일 강제 X, "올리면 자란다" 느슨한 보상. 풀 SNS는 5/11 보류 결정 존중.
